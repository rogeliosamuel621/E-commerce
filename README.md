## Getting started

### Install
```
npm install:app
```

### Development
- api
```
cd packages/api && npm dev
```
- frontend
```
cd packages/frontend && npm dev
```

### Production
```
npm run build:app
npm run start:app
```

### Test
```
npm run test:app
```
